# nucleo-f303k8-16x2-lcd

16x2 LCD Library for NUCLEO-F303K8

> [!NOTE]
> This repository serves as a minimal example to showcase the LCD library using STM32CubeIDE.

## Installation

1. Copy the [lcd.h](https://github.com/ostromia/nucleo-f303k8-16x2-lcd/blob/main/Core/Inc/lcd.h) file into your `Inc` directory.
2. Copy the [lcd.c](https://github.com/ostromia/nucleo-f303k8-16x2-lcd/blob/main/Core/Src/lcd.c) file into your `Src` directory.
3. Include `#include "lcd.h"` within your [`main.c`](https://github.com/ostromia/nucleo-f303k8-16x2-lcd/blob/2eb803a0310cf0ca8c375cf3effd988da9077f3c/Core/Src/main.c#L24) file.
